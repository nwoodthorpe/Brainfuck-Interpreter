# Brainfuck-Interpreter
A Brainfuck Interpreter written in Ruby. 

Reads one the program as one line from stdin. Comma commands are input from stdin. Outputs to stdout in ASCII format.

Sample usage:

```
ruby interpreter.rb 
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++[>+>+>+>+<<<<-]>+++.>++++++++++++++++++.>.>++++++++.>++++++++++.
```
